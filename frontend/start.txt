# PDPL Compliance Assessment Platform — Frontend Setup

## Prerequisites
- Node.js 18+
- npm

## First-time setup

1. Install dependencies:
     npm install

2. Copy env file:
     copy .env.example .env.local
   (Default API URL is http://localhost:8000)

## Running the frontend

     npm run dev

   App: http://localhost:3000

## Routes

  /              → Landing page ("Check Your PDPL Compliance in 2 Minutes")
  /assessment    → Multi-step assessment form (Sections A, B, C)
  /assessment?demo=true  → Pre-filled demo data
  /signup        → Signup gate (shown after assessment)
  /results/[id]  → Results dashboard (RAG score + gaps + AI summary + CTAs)
